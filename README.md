# Portfolio de Proyectos

Este repositorio contiene una colección de proyectos académicos y personales desarrollados durante el Grado en Ciencia e Ingeniería de Datos.

## Proyectos

### 1. Proyecto PLNE (`proyecto_plne`)
**Descripción:** Compilación de corpus y procesamiento léxico. Utiliza técnicas de Procesamiento del Lenguaje Natural (PLN) y modelos como BM25.
**Tecnologías:** Python, Jupyter Notebook, NLTK, Scikit-learn.

### 2. Análisis de Datos ACB (`proyecto_analisis_acb`)
**Descripción:** Análisis estadístico y visualización de datos de la liga de baloncesto ACB.
**Tecnologías:** R / Python (según corresponda), Data Analysis.

### 3. Infraestructura Web (`proyecto_infra_web`)
**Descripción:** Proyecto relacionado con el despliegue y gestión de infraestructuras web.
**Tecnologías:** Servidores, Cloud, Despliegue.

### 4. Clasificación de Piezas con Deep Learning (`proyecto_aiva`)
**Descripción:** Inspección de pequeño material mecánico (tornillos, tuercas) utilizando técnicas de Deep Learning sobre el dataset MVTec Screws.
**Tecnologías:** Python, Deep Learning, Computer Vision.

## Estructura del Repositorio
Cada proyecto se encuentra en su propia carpeta con la siguiente estructura general:
*   `src/`: Código fuente y notebooks.
*   `data/`: Datos utilizados (comprimidos).
*   `README.md`: Documentación específica del proyecto.
