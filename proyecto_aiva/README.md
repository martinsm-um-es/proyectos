# TF02. Clasificación de piezas mediante Deep-learning

## Autor
**Martín Solano Martínez**

## Descripción
En este trabajo desarrollaremos una aplicación software en Python para la inspección de pequeño material mecánico utilizando técnicas de deep-learning para su clasificación.

El conjunto de datos utilizado es **MVTec Screws V1.0**.

## Estructura del Proyecto

*   **`src/`**: Contiene el código fuente y el notebook principal (`main.ipynb`).
*   **`data/`**: Contiene los datos del proyecto comprimidos (`TF_AIVA_datos.zip`). Nota: Este archivo (3.7GB) no se incluye en el repositorio por límites de tamaño.
*   **`models/`**: Modelos entrenados y archivos relacionados.
*   **`TF02_AIVA.pdf`**: Memoria del proyecto.
*   **`Presentación_AIVA.pdf`**: Diapositivas de la presentación.
*   **`video_AIVA.mp4`**: Video demostrativo.
